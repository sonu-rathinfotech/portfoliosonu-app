## Two Learning Tracks

**80/20 Core** (20 topics) — The most frequently tested concepts. Covers ~80% of system design interviews.

**0-to-100 Deep Mastery** (42 topics) — Full production-depth coverage for senior engineers and real-world architecture.

## Each Topic Includes

Every topic follows a consistent 13-section deep-dive template:

1. Origin story — who built it, why, and what was failing
2. What existed before — the predecessor technologies
3. The problem it solves — formal and practical
4. Real-world implementation — how FAANG companies use it
5. Deployment and operations — how it runs in production
6. Analogy — a memorable non-technical explanation
7. Mental models — frameworks for remembering and applying
8. Challenges and failure modes — what goes wrong
9. Trade-offs — no free lunch analysis
10. Interview questions — beginner, mid, and senior tiers
11. Code examples — pseudocode + Node.js with line-by-line explanation
12. Bridge to next topic — connecting concepts into a learning path

## Recommended Learning Paths

**Interview Sprint (4-6 weeks):** 20 core topics + top case studies

**Full Foundations (8-12 weeks):** All 80/20 Core topics in order, then deep mastery by chapter

**Senior Deep Track (12-16 weeks):** All 62 topics with extra time on distributed systems and senior-level designs
